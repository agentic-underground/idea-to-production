# Adopt comfyui-mcp in publish (handler-comfyui switches curl → MCP)

**Brief**: `handler-comfyui` currently talks raw HTTP to a live ComfyUI server (the LAN is the only trust
boundary — a known Phase-0 gap). Once the secured `comfyui-mcp` server exists, publish's graphical
value-handler switches from `curl` to the `mcp__comfyui__*` tools, so generative figures flow through the
allowlisted, validated, authenticated surface — and declines gracefully when the server is absent.

## User stories
- AS a publish illustrator I WANT generative raster figures to go through the secured comfyui-mcp SO THAT I
  am not depending on an unauthenticated LAN service with arbitrary workflow execution.
- AS the marketplace owner I WANT handler-comfyui to degrade gracefully when comfyui-mcp is absent SO THAT
  the illustrator still falls back to vector handlers.

## EARS specification
- `EARS-001` (Ubiquitous): THE SYSTEM SHALL drive ComfyUI generation through `mcp__comfyui__*` verbs when
  the comfyui-mcp server is available, never raw `curl` to a ComfyUI HTTP port.
- `EARS-002` (Event-driven): WHEN handler-comfyui needs a raster figure THE SYSTEM SHALL submit an
  allowlisted workflow template via comfyui-mcp and poll for the result through the MCP surface.
- `EARS-003` (Unwanted): IF the comfyui-mcp server is unreachable THEN THE SYSTEM SHALL decline the
  generative slot (so the orchestrator fills it with a vector option), never fall back to an
  unauthenticated raw-HTTP call.

## Acceptance criteria
1. Given comfyui-mcp is running, When the illustrator requests a generative figure, Then handler-comfyui
   uses `mcp__comfyui__*` tools and no `curl` to a ComfyUI port appears in its trace.
2. Given comfyui-mcp is unreachable, When a generative figure is requested, Then handler-comfyui declines
   and the illustrator selects a vector handler instead, with the degrade disclosed.

## Human interface test plan
None — handler-comfyui is an agent value-handler, not a UI element. Proof: with the server up, its trace
uses `mcp__comfyui__*` and contains no `curl` to a ComfyUI port; with it unreachable, it declines and the
degrade is disclosed (asserted on the handler's behaviour against a mocked MCP surface).
