## Shared infrastructure
- **The shared exclusion + severity config (lands first).** PLAN_0030 establishes the unified
  `.securityignore` (per-lens sections) that every lens reads; PLAN_0031 establishes the
  `security.config.json` severity→verdict policy the gate applies. The scanners and new lenses below
  consume both — so these two PLANs are ordered first (`order` 0000/0001) and own that shared infra; later
  PLANs never re-create it.
- **The gate verdict-synthesis surface** — `/security:scan-all` (the gate) computes one verdict from all
  lens findings. The new lenses (license-audit PLAN_0032, SBOM PLAN_0033, container/IaC PLAN_0035) fold
  into that synthesis; diff-scoped mode (PLAN_0036), live-credential verification (PLAN_0037) and the
  findings ledger (PLAN_0038) wrap/extend the same gate. The ecosystem/manifest detection from
  `scan-dependencies` is reused by license-audit + SBOM.
- **Rename caveat (load-bearing for the builder).** Legacy items name the plugin **SENTINEL** with paths
  `plugins/sentinel/` and the verbs `/security-gate`, `dependency-audit`, `pii-audit`, `secret-scan`. The
  plugin is now **`security`** (`plugins/security/`), and those map to `/security:scan-all`,
  `/security:scan-dependencies`, `/security:scan-for-pii`, `/security:scan-for-secrets`. The config file is
  `security.config.json` (legacy `sentinel.config.json`). Target the current `security` plugin throughout.

## Definition of done
- All nine PLANs are `completed` (the engine rolls EPIC completion up from its PLANs).
- `.pipeline/verify` is GREEN for each landed PLAN; `scripts/verify-prereqs.sh` (incl. check P, the
  security `check` manifest's well-formedness B/G, and check J four-mirror for any new skill) stays GREEN.

## Execution contract (for the builder)
The engine builds one PLAN at a time in `## Plans` order, running each PLAN's `## Construction process`
(FOUNDRY scoped to the `(EPIC, PLAN)` pair). The shared-config PLANs (0030/0031) land first. A failed PLAN
leaves its row `engaged` and resumes next tick; the EPIC lands only when all nine PLANs are GREEN.
