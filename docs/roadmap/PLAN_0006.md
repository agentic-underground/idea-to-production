- **Rename SENTINEL → security.** Every legacy item names the plugin "SENTINEL" with paths
  `plugins/sentinel/` and verbs `/security-gate`, `dependency-audit`, `pii-audit`, `secret-scan`; the
  plugin is now `security` (`plugins/security/`) with `/security:scan-all`, `/security:scan-dependencies`,

  `sentinel.config.json`). Every PLAN carries this caveat for the builder.
- **Dependency order** (item 63 `depends_on: #60`) is preserved as PLAN_0034 `depends_on: PLAN_0033`; the
  shared `.securityignore` (PLAN_0030) and severity policy (PLAN_0031) are ordered first as the EPIC's
  shared infrastructure.
