# Sources — the naming canon's provenance

The art-of-naming canon in this directory is distilled from the following. Re-consult when extending a
vein, a framework, or the affix catalogue; add new sources as the canon grows (self-improvement
covenant).

## Foundational frameworks
- Marty Neumeier — *The Brand Gap* / *Zag*; "Strong vs. Weak Names" — https://www.martyneumeier.com/
- Alexandra Watkins — *Hello, My Name Is Awesome* (SMILE & SCRATCH)
- Igor International — Naming Guide & Tools — https://www.igorinternational.com/process/naming-tools-name-creation.php
- Catchword — Branding Naming Glossary — https://catchwordbranding.com/glossary/

## Taxonomy & classification
- How Brands Are Built — Types of Brand Names — https://howbrandsarebuilt.com/types-of-brand-names/
- Nancy Friedman (Fritinancy) — The Five Types of Brand Names — https://wordworking.medium.com/the-five-types-of-brand-names-41e51fef8ae3
- ResearchGate — A Comprehensive Framework of Brand Name Classification — https://www.researchgate.net/publication/276547620
