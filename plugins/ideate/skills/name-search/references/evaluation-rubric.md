# Evaluation rubric — the scored adversarial challenge

The **challenge** phase. Availability (the script's verdict) tells you a name *can* be used; this tells
you whether it *should* be. The two are **different verdicts in different columns** — never conflate a
name killed for being *taken* with one *demoted in the challenge* (the report's cardinal rule).

**Default `survives = false`.** A name earns survival; it is not assumed. Score each survivor; the
scores drive the ranked shortlist.

## The axes

Grounded in [`../../../knowledge/naming/frameworks.md`](../../../knowledge/naming/frameworks.md) and
[`../../../knowledge/naming/phonosemantics.md`](../../../knowledge/naming/phonosemantics.md).

1. **Neumeier 7** (score each 1–5; report the weak ones):
   distinctiveness · brevity (≤4 syll) · appropriateness · easy spell/say · likability/mouth-feel ·
   legs/extendability · protectability.
2. **SMILE present?** — is it Suggestive · Memorable · Imagery · Legs · Emotional? (note which it has).
3. **SCRATCH flags** — each present flag is a demerit, cite it:
   Spelling-challenged · Copycat · Restrictive · Annoying · Tame · Curse-of-knowledge · Hard-to-pronounce.
4. **Archetype-fit** — does it land the charter's chosen archetype(s)? (1–5)
5. **Sound-symbolism-fit** — does the *sound* match the archetype palette, not just the meaning? (1–5)
6. **Cross-language connotation** — cross-reference the script's `connotationFlags`; plus any awkward
   meaning/slang/homophone in a charter-relevant language. Flag, with the language.
7. **Hidden collision / trademark adjacency** — a live entity or near-mark the registries can't see
   (web-search caveat, never asserted as cleared); cross-reference the script's `neighbors` (typo-squat
   proximity) and `domains`.

## Composite scoring

Report three headline scores per survivor (1–5), as the shortlist expects:
**philosophy-fit** (values + archetype + appropriateness) · **distinctiveness** (Neumeier-1 +
protectability + neighbour-cleanliness) · **sayability** (Neumeier-4/5 + sound-symbolism − SCRATCH
pronunciation/spelling flags).

## Honesty rules (carried from the skill)

- **availability ≠ challenge** — separate columns, separate language.
- **Don't self-clear trademark or connotation** — these are flags + caveats, not clearances; recommend
  formal USPTO/EUIPO search before commitment.
- **A name can be CLEAR (available) and still be killed here** — e.g. Tame, Hard-to-pronounce, or an
  archetype mismatch. Say *why*, on which axis.
